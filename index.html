Angela Barriga Rodriguez

PhD Research Fellow

Software Engineer

Contact:

Hello! I'm Angela :) Currently, I work as a PhD research fellow in the Western Norway University of Applied Sciences (HVL), based in the beautiful city of Bergen, Norway.

My main current research topic is applying reinforcement learning methods to repair software models (read more about it here). I received a double master's degree in Computer Engineering and IT management (award for best academic record) from the Universidad de Extremadura (UNEX) (Caceres, Spain) in 2017 and the bachelor's degree in Software Engineering from the same institution in 2015.

So far, my professional experience is diverse: a mix of technical and academic jobs. In the past, I have worked as a full stack web developer, researcher and teaching assistant at master's level. I love to face challenges and I put passion in everything I do.

Projects I'm proud of


Here you can find some of the most interesting projects I have developed.

  • Personalized and Automatic Repair of MOdels using REinforcement Learning (present): PhD thesis project developed in the Department of Software Engineering, Sensor Networks and Engineering Computing. On-going project to repair broken software models using reinforcment learning methods. The project is built in Java and, at the moment, it includes a Q-Learning and Transfer Learning implementation. More information here.
  • Reinforcement Learner Through Observation (2018): Hackathon project developed in the MIT event: Media Lab Berlin - Signal and noise, as part of the Machine learning for creative AI track. The project consisted on a Deep Q Learning algorithm that only learnt when a user is in front of the computer where it is executed. For this, we used a face detector to conditionate the algorithm's learning. The project is built in Python. You can fin it in this GitHub repository.
  • A Visual Framework for Transparent and Accessible Machine Learning (2018): Research project focused on a visual interface for enhancing Machine Learning accessibility and transparency. Exploring the possibilities of XAI (Explanaible Artificial Intelligence). The project was built using Django and Scikit. You can get more information about the project in this workshop paper and this demo link.
  • Situational Context: Distributed management of contextual information (2017): Master's Thesis developed at the Social and Pervasive Innovation Lab (SPILAB) in the UNEX. Project focused on the description of a theoretical framework for building Internet of Things systems using smartphones, taking into account psychological theories about human-computer interaction and communication to create human-centric environments. You can download the original thesis here. This project got a special distinction award.
  • Smart System for Behaviour Detection (2015): Bachelor's thesis developed at Quercus Software Engineering Group in the UNEX. Project focused on identifying different body poses and sudden falls by using neural networks and 3D cameras, built with Java. You can find the original thesis (in Spanish) here, and a jornal paper about the project here. This project got a special distinction award.
